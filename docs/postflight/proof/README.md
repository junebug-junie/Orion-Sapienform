# Answer-Depth Proof Bundle (Pass 3)

This directory contains concrete, proof-oriented evidence generated by the Pass 3 harnesses.

## Canonical evidence artifacts

1. Discord golden-path delivery prompt
   - `discord_deploy_golden_path_evidence.json`
   - `discord_deploy_golden_path_evidence.md`

2. Supervisor meta-plan finalization proof
   - `supervisor_meta_plan_finalize_evidence.json`
   - `supervisor_meta_plan_finalize_evidence.md`

## What is and is not “real”

- The harnesses **do not** call external LLMs / real PlannerReact over the bus.
- They **do** exercise the **real runtime wiring**:
  - Cortex exec `Supervisor` control flow
  - Agent-chain pack merge + tool resolution (`delivery_pack`)
  - triage hard cap after step 0
  - repeated `plan_action` escalation path
  - finalization via `finalize_response`
  - meta-plan rewrite gating for instructional modes

## How to regenerate evidence

Run the canonical suite runner:

```bash
python scripts/run_answer_depth_proof_suite.py
```

The runner sets `ORION_PROOF_WRITE=1`, so the harnesses overwrite evidence files here.
