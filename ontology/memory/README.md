# Orion memory ontology (GraphDB)

Load `orionmem-v2026-05.ttl` and `shapes-orionmem-v2026-05.ttl` in GraphDB **Workbench** via **Import** → **RDF** → choose file(s), target repository, then import. SHACL shapes can live in the same repository as data or a dedicated shapes graph if your deployment separates validation graphs.
